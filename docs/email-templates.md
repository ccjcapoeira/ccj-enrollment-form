# 自動返信メール文面

このファイルは、`google-apps-script.js` に設定されている自動返信メールの内容を、確認・編集しやすい形に整理したものです。

実際に送信される文面を変更する場合は、`google-apps-script.js` の該当メール本文も更新してください。

### 口座振替を選択した場合に追加される案内

支払い方法に「口座振替」が含まれる場合、上記本文の申し込み内容の下に次の案内が追加されます。

```text
【口座振替ご登録のお願い】
口座振替のお手続きについては、以下のページをご確認ください。
https://www.nss-jp2.com/page_ex.jsp?CONTROLID=KTS0960&BUSINESSID=initDisp&DISPLAY_KEY_param=2k7XiZXZspuiZX

お手続き時の団体コード（必須）: 0944082

別途ご提出いただく書類はございません。
引き落としは毎月27日前後（月末付近）を目安として行われます。
お手続きはできるだけお早めに完了ください。完了のタイミングにより、初回の引き
落としが翌月以降となる場合があります。
```

### 備考

- 予備電話番号、予備メールアドレス、保護者氏名、備考は、入力がある場合のみメールに表示されます。
- 規約同意、個人情報同意が空の場合は、メール上では「同意」と表示されます。
- 初月会費と初回合計は、選択されたコースに応じて次の金額を表示します。
  - レギュラー: 初月会費 6,700円 / 初回合計 10,200円
  - アドバンス: 初月会費 9,400円 / 初回合計 12,900円
  - プロ: 初月会費 13,000円 / 初回合計 16,500円
  - キッズ: 初月会費 5,000円 / 初回合計 8,500円
  - ドロップイン: 参加費 2,500円 / 回
