# 退会フォーム（`leave-request.html`）読みやすい版

このページは、`leave-request.html` の内容を運用者向けに文章化したものです。

## 画面タイトルと案内文

- タイトル: **退会のお手続き**
- 冒頭案内:
  - 「退会をご希望の方は、以下のフォームをご入力ください。送信をもって手続きは完了です。」
- 注意表示:
  - 「本フォームの送信をもって退会手続きは完了します。」
  - 「電話・メールでの追加手続きは不要です。」

## 入力項目

### 必須

- 氏名
- 電話番号
- メールアドレス
- 退会希望月（`type="month"`）

### 任意

- 最終参加予定日（`type="date"`）
- 退会理由（自由記述）
- 備考（自由記述）

## バリデーション

- 必須項目が空の場合はエラー表示
- メールアドレスは基本的な形式チェックあり
- 電話番号は数字換算で **10桁または11桁** を許容

## 送信時の動き

1. 「退会のお手続きを送信する」ボタン押下
2. バリデーション通過後、Google Apps Script の Web アプリに `POST` 送信
3. 成功時はフォームを非表示にして完了画面を表示
4. 失敗時は「送信に失敗しました。時間を置いて再度お試しください。」を表示

送信データは `formType: "leave_request"` として送られます。

## 完了画面の文言

- 見出し: **退会のお手続きが完了しました**
- 本文:
  - 「退会のお手続きを受け付けました。内容確認が必要な場合のみ、担当よりご連絡いたします。」

## 自動返信メールについて

- 現在の実装では、`leave-request` 送信後に会員向け自動返信メールは送っていません。
- スプレッドシートへの保存は行います（GAS 側 `handleLeaveRequest_`）。
